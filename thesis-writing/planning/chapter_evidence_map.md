## Planned Causal Claim Template

Any causal-result claim must include:

| field | required content |
| --- | --- |
| exposure/treatment | A named `LAT_*` proxy-state exposure from the selected dataset/run. |
| outcome | `in_hospital_mortality`. |
| target estimand | Use "mean estimated CATE over analyzed rows" or "matched-pair outcome difference" unless advisor approves ATE/ATT wording. |
| adjustment assumptions | Explicit DAG-derived observed adjustment set and missing graph candidates. |
| overlap assumption | Matching support/indirect support diagnostics; dedicated overlap plot if generated later. |
| sensitivity limitation | RV/benchmark/permutation diagnostics if non-PFN; PFN lacks these by design. |
| intervention-definition concern | State that many proxy states are illness/exposure constructs rather than well-defined interventions. |
| evidence status | Result-confirmed only after selected source row/table is validated. |
