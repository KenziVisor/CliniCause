# 17. Evaluation and Result-Selection Policy

Chapter 9 should state the policy that Stage 4.6A must apply before numerical drafting.

A result may be admitted to the Results chapter only when:

1. the producing run is identified;
2. dataset, estimator/model, and sampling condition are known;
3. the source artifact exists;
4. the required schema is validated;
5. the result status is not failed;
6. source fields are interpreted correctly;
7. the input and output relationship is documented;
8. any fallback diagnostic is labeled as fallback;
