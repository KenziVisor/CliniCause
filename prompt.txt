Implement argparse flags for the router itself.

Required flags:

--dataset {physionet,mimic,both}

Which dataset to run.
If "both", run preprocessing/tagging/plots for both datasets and prepare both datasets for STraTS.

Important:
The current run_full_main.sh script runs both PhysioNet and MIMIC internally.
Therefore, for --run-strats true, the safest mode is --dataset both.
If --dataset is physionet or mimic and run-strats is requested, fail clearly unless --allow-existing-strats-inputs is provided and the other dataset files already exist in the STraTS repo.

--run-id <str>

Logical run id for the unified run directory.

  exp_001
  debug_physionet
  full_run_2026_06_28

--output-root <path>

  runs

--thesis-repo-root <path>

  current working directory

--strats-repo-root <path>

Path to STraTS repo root.
Required unless --run-strats false and no STraTS stages are requested.

--strats-script-path <path>

Path to STraTS shell runner relative to strats repo root or absolute.

  run_full_main.sh

Supported values:
  preprocessing
  tagging
  trees
  prepare-strats
  run-strats
  collect-strats
  normalize-predictions

Print commands and planned file operations without executing them.

Validate paths, dataset mapping, stage dependencies, and planned inputs/outputs, then exit.

  sys.executable
