# 25. Blocking rules

Use:

```text

```

if:

* a figure hash changed unexpectedly;
* a source file is missing;
* a numerical figure cannot be reproduced;
* regenerated pixels differ without explanation;
* a figure includes the wrong rows;
* an axis, sign, value, label, title, or legend is wrong;
* PhysioNet shock is hidden or sign-flipped;
* a ranking uses an unauthorized estimator or aggregation;
* a DAG copy does not match its archived canonical source;
* the active-source graph and archived pickle differ materially;
* the MIMIC hash-name collision remains unresolved;
* a caption claims exact producing provenance not established by evidence;
* an excluded figure entered the thesis;
* a table contains an unreconciled value;
* a failed or missing result is displayed as zero;
* a robustness result is presented as primary;
* a reproducibility artifact contains an unreconciled hash or status;
* the thesis fails to build;
* protected evidence changed.

```text

```

only when:

* every result figure is source-exact;
* every result figure is byte- or pixel-equivalent to isolated regeneration;
* every DAG is an exact canonical copy;

* the MIMIC duplicate-name issue has a supported explanation;

* every appendix claim is supported;
* reproducibility records are internally consistent;
* all protected files are unchanged;
* the thesis builds;
